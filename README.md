## Installation

```bash
$ npm install
$ npm i -g @nestjs/cli
```

## Running the app
### via docker
```bash
$ docker-compose up
```

### via system
```bash
$ nest start --watch <microservice>

```
*this requiries to have **@nestjs/cli** installed*

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
